### Project Overview
The sushi place that my girlfriend and I got to all the time has a bad website built by someone who wanted their money for low effort. I want to create a better website for them and the give it to them!

### Stack
- React 
- React Router
- CSS
- Google Maps Embed
- Node.js 

## Run the Project

In the project directory, you can run:

`npm start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

The page will reload when you make changes.\
You may also see any lint errors in the console.
